# Course Computer Enviroments 
An environment refers to the specific configuration in which your code and tools are executed. Most of them are installed in CONDA.
Command line to activate or deactivate an enviroments. 

```bash
conda activate <env_name>   # activate enviroments      
conda deactivate            # deactivate 
```
